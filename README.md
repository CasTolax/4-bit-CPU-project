# 4-bit-CPU-project

Tübitak projem olan ve merakımın sınırı zorladığım proje, şu ana kadar yaptığımız en zor projelerdem biri. Mühendislik için bu yaş çok erken olabilir ama benim bu yaşta mühendis gibi düşünmemi engelleyen bir şey yok.

# AÇIKLAMA/DESCRİPTİON
Yapılan bu projede amaç, gerçek bir işlemcinin nasıl çalıştığını,frekans,saat frekansı,bilgi işlemi ve bilginin saklanması, mantık devreleri kurup olayları ve çıktıları gözlemlemektir. yani hemen hemen araştırma gibi düşünülebilir. Daha önce C++'dan 4-bitlik bir işlemciyi simüle edebilmiştim. buradan ulaşabilirsiniz: https://github.com/CasTolax/4-bit_cpu

Projeyi 74HC entegreleriyle yaptık ve oldukça da zor olduğunu söyleyebilirim. Projeye başlanmak isteniyor ise kesinlikle Ben Eater youtube kanalından bilgi veya öğrenmeye çalışın,faydasını göreceksiniz.

The goal of this project is to understand how a real processor works—including frequency, clock speed, data processing, and data storage—by building logic circuits and observing the events and outputs. In other words, it can be thought of as a research project. I had previously simulated a 4-bit processor using C++. You can access it here: https://github.com/CasTolax/4-bit_cpu

We built the project using 74HC integrated circuits, and I can tell you it was quite challenging. If you’re thinking of starting this project, I strongly recommend checking out the Ben Eater YouTube channel for information or to learn more—you’ll find it very helpful.
